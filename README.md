# QGIS Geospatial Analysis Portfolio

This repository showcases sample geospatial analysis projects built using QGIS. The focus of these projects is spatial data processing, digital mapping, and geospatial analysis used for environmental, urban planning, and infrastructure research.

## Tools & Technologies
- QGIS
- Spatial Data Analysis
- Raster and Vector Data Processing
- Geospatial Data Visualization
- Coordinate Reference Systems (CRS)
- Spatial Joins and Buffer Analysis



## Project 1: Urban Accessibility Analysis

This project analyzes accessibility to public facilities in urban areas using spatial buffers and distance analysis.

Key Components:
- Road network mapping
- Buffer zone analysis
- Accessibility heatmap creation
- Spatial joins for facility coverage



## Project 2: Environmental Land Use Mapping

A geospatial project analyzing land use patterns using raster and vector data layers.

Key Components:
- Satellite imagery interpretation
- Land classification mapping
- Spatial overlay analysis
- Land use change visualization



## Project 3: Infrastructure Proximity Analysis

This project evaluates proximity between infrastructure assets and residential areas using spatial queries.

Key Components:
- Vector layer analysis
- Distance calculations
- Infrastructure risk mapping
- Geospatial visualization



These projects demonstrate the use of QGIS for geospatial data analysis, spatial modeling, and digital mapping workflows used in research and AI training datasets.
